Desurvey Method
Select how to convert downhole depth measurements into 3D XYZ coordinates along the drillhole trace.
MINIMUM CURVATURE
Recommended

Assumes smooth arc between survey stations using the ratio factor. Industry standard for modern resource estimation. Best for curved and deviated holes.

TANGENTIAL
Simple

Projects using dip/azimuth at the start of each survey interval. Fast. Suitable for near-vertical holes. Accumulates error in deviated holes.

BALANCED TANGENTIAL
Classic

Averages direction vectors at start and end of each segment. A good compromise between accuracy and computational simplicity.

💡Minimum Curvature: RF = 2/dl × tan(dl/2). For each segment, a smooth arc is computed between consecutive survey stations.

① Drillhole Spacing Rule
Block size should be 40–60% of the average 2D drillhole spacing. Ensures adequate data density for each block and prevents over-smoothing of grade estimates.
—
avg spacing × 0.5

② Variogram Range Criterion
Block size should be no more than 1/3 of the variogram range (Journel & Huijbregts, 1978). Ensures multiple blocks fit within the zone of spatial correlation. Enter a fitted range in Step 4 for best results.
—
range ÷ 3

③ Composite-to-Block Ratio
Optimal estimation when composite length is 0.5–1.0× the block size in the Z direction (Deutsch, RMS 2022). Constrains the vertical cell size independently of horizontal spacing.
—

④ Kriging Neighbourhood Adequacy
Each block should be estimable from 8–24 composites in its kriging neighbourhood (Webster & Oliver). Too few samples produce unreliable estimates; too many over-smooth.
